Naver Demo

A jQuery plugin for responsive navigation. Learn More.

Basic

Naver will atuomatically turn a basic navigation system into a mobile friendly system:

$("nav").naver();
<nav>
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
</nav>

Animated

The navigation states can be also be animated. Note: This uses CSS transition which can be slow on some mobile devices.

$("nav").naver({
    animated: true
});

Reset

Naver works by setting heights, so when transitioning between break points you may wish to reset the navigation state. You can also use a plugin like Rubberband to detect and act on break point changes instead of general resize events:

$("nav").naver();

$(window).on("resize", function(e) {
    $(".naver").trigger("close.naver");
});
